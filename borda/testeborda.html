Teste de borda

Para fazer essa borda faz-se as seguintes declarações em CSS:

h1 {
    border: 12px solid black; //faz a borda
    padding: 10px; 
    border-image-source: url('../midia/bordas.png'); //adiciona a imagem desejada à borda
    border-image-slice: 27; //define como a borda será 
    border-image-repeat: repeat; //indica que é para a borda se repetir
}

Shorthand

border-image: url('../midia/bordas.png') 27 repeat;
