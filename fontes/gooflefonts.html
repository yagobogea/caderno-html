Google Fonts

Caso se sinta limitado à fontes padrão que vem no computador, há como inserir fontes externas à partir do Google Fonts. Dessa maneira, coloca-se a mais no código a fonte desejada. Para fazer isso, basta entrar no link do google fonts e selecionar a fonte desejada, escolher @import na parte de embed code e inserir na parte de CSS do código, se for CSS dentro do HTML coloca-se dentro da tag head e se for CSS externo coloca-se acima de todos os seletores.
