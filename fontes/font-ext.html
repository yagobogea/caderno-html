Para utilizar fontes externas, necessita-se de ter o arquivo baixado no seu computador, o arquivo pode ser baixado de vários sites, um exemplo é o DaFont, ao ter o arquivo baixado o insira na pasta que contem o seu index. Para inserir a fonte nos textos, precisa-se criar um Font-Face personalizado dentro do CSS, como aqueles que usam o @ visto na aula passada,

OBS: os arquivos serão salvos ou ttf ou otf, depende do navegador para usá-los. Indica-se colocar os dois para que se um não funcione, o outro funcione. Ao se inserir o arquivo nas CSS, deve-se colocar um format(); que podem ser:

  • opentype (normalmente usado para otf)
  • truetype (normalmente usado para ttf)
  • embedded-opentype
  • truetype-aat (Apple Advanced Typography)
  • svg
